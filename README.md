# sketch-ui

## Project setup (SETP 1)
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### run backend python server (where save and solve the data retrieved from ui) (STEP 2)
```
python app.py
```
